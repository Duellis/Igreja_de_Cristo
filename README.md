# ⛪ Portal Igreja de Cristo

Uma plataforma web completa desenvolvida para centralizar materiais de estudo, louvor e comunicação da congregação, facilitando o acesso à informação e a integração de novos membros e visitantes.

## 🌐 Acesso Online

O portal está no ar e pronto para uso! Não é necessário baixar ou instalar nenhum arquivo. Acesse a plataforma diretamente pelo link abaixo:

🔗 **[Acesse o Site Aqui](https://duellis.github.io/Igreja_de_Cristo/)**

---

## 🚀 Funcionalidades

* **📖 Acervo Histórico:** Espaço dedicado para preservar e compartilhar a história da congregação.
* **🎵 Hinário Digital Interativo:** Catálogo completo de hinos equipado com um motor de busca dinâmico e um tocador de áudio automático integrado.
* **📚 Estudos e Artigos:** Área para a disponibilização de textos, mensagens e materiais de aprofundamento.
* **📌 Mural de Recados:** Quadro virtual para comunicação, avisos e interação.
* **📸 Galeria Multimídia:** Acervo de registros visuais, contendo vídeos e fotos dos encontros e eventos.
* **📝 Cadastro de Interessados:** Formulário intuitivo para captação de pessoas que desejam participar de estudos e pregações.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização visual e responsividade das páginas.
* **JavaScript:** Lógica de programação para o motor de busca do hinário, funcionamento do player de áudio e interatividade geral do site.
